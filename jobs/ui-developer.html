Web / User Interface Developer (Junior and Senior)

Overview

Siku-Siku started as a web user interface development company. We design websites, build custom interfaces (e.g. web form inputs / controls), code newsletters, and customize content management system (CMS) themes. All of these with the main focus on accessibility, usability, and user experience. So, our user-friendly deliverables are — and must be — compatible across all targeted browsers (and email clients, in case of newsletter); with the main goal of providing the best possible user experience. We are also steadily moving into the mobile world, where accessibility, usability, and user experience are as important as, if not more than, on the web. In terms of the team, we are a small group of HTML/CSS/JavaScript experts, graphic designers, and user interface / experience (UI/UX) designers. Our clients are startup companies in the Bay Area, California.

We are currently looking to add a UI Developer to join our new team in Jakarta (we consciously keep our team small). This is a specialized position that requires a full depth understanding of any related front-end web engineering. It’ll be a great exposure to work directly with some Silicon Valley startups.

Responsibilities:

  1. Implement web pages from scratch with HTML / CSS / JavaScript. The design will be provided in Photoshop comps.
  2. Create customized user interface (UI) elements e.g. pull-down menu, radio button, lightbox. This requires strong understanding of HTML / CSS / JavaScript.
  3. Fix bugs that are related to HTML / CSS / JavaScript and browser incompatibility, which might have been written by other developers or legacy code.
  4. Document (as a blog post) any interesting findings while working on those three items above.

Required Experience:

  1. Strong understanding of HTML / CSS / JavaScript (jQuery): produce web pages exactly as designed in the PSD comp.
  2. Deep understanding of browser technologies and limitations (e.g. rounded corners in IE, CSS3 support, etc.).
  3. Versed in CSS techniques and tricks (e.g. sliding doors with transparent background, clearfix hack, sprite, etc.).
  4. Thorough knowledge of newsletter implementation (e.g. which constructs are “safe” for majority of email clients).
  5. Skillful in generating web-ready images from PSD files.

Desired Experience:

  1. Familiarity with version control system (we use git).
  2. WordPress theme development experience.
  3. Understanding and appreciation of web usability and accessibility.

Skills:

  1. Excellent problem solving and debugging skills.
  2. Proven ability to deliver projects on time.
  3. Good communication skills in English, both verbal and written.
  4. Willingness to learn and to teach.
  5. Design skill is a big plus.

Education:

BS in Computer Science or equivalent.

Compensation:

Competitive, based on experience.

How to Apply:

Send an email to jobs [at] siku-siku.com and include the following:

  1. Latest CV / resume
  2. Most recent photo (social network profiles are also acceptable)
  3. Contact info (phone and/or email)
  4. “UI Developer” as the email subject
  5. Indicate whether you’re applying for the Junior or Senior position, depending on your self-assessed experience
  6. Expected salary (optional)

Notes:

During the interview, you are required to demonstrate that you have the aforementioned experience. For example:

  1. When do you (not) use HTML tables?
  2. HTML: What are the differences between quirks and strict mode? What are their implications to your design?
  3. HTML: When do you use <h1> vs. <p> vs. <span> vs. <div>?
  4. CSS: When do you use float vs. absolute positioning?
  5. CSS: When do you need to set the z-index?
  6. jQuery: What are the difference between the built-in JavaScript this keyword and $(this)?
  7. jQuery: Write a simple script to show / hide a DOM element every 2 seconds.
